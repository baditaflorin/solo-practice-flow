# Contributing

Thanks for improving Solo Practice Flow.

## Local Setup

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `make install-hooks`.
4. Run `make dev`.

## Commit Style

Use Conventional Commits, for example:

```text
feat: add invoice export
fix: handle empty lead list
docs: explain Pages deployment
```

## Checks

Run `make test`, `make lint`, and `make smoke` before pushing.
