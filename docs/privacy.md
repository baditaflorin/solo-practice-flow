# Privacy

Solo Practice Flow has no analytics in v1.

Client records are stored in the browser's IndexedDB for the site:

https://baditaflorin.github.io/solo-practice-flow/

The app does not send leads, proposals, contracts, signatures, invoices, or exports to a hosted Solo Practice Flow server. Optional local LLM generation calls only the endpoint the user configures in the app. Exported files are downloaded by the browser and are the user's responsibility to store safely.
